You can push the code directly to your GitHub account using the provided script:

./push-to-github.sh

This script will create a new repository and push all the code to it.

Setting Up Locally

After downloading the code:

  1. Extract the ZIP file
  2. Install dependencies: 
    npm install
  3. Set up a PostgreSQL database
  4. Copy the .env.example file to .env and update the database connection details
  5. Run the database migrations: 
    npm run db:push
  6. Start the development server: 
    npm run dev

Note: To set up the SendGrid email integration, you'll need to provide your own SENDGRID_API_KEY in the .env file.
